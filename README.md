# B'z セットリストアプリ（B'z Live Setlist DB）

このアプリは、**B'zのライブセットリストを見やすく管理・比較するためのシンプルなWebアプリ**です。  
ブラウザだけで動き、インストール不要で使えます。

---

## 1. このアプリでできること

このアプリでは、主に次のことができます。

- **公演ごとのセットリスト確認**  
  各公演の日付・会場・曲順を一覧で見られます。
- **曲別ランキング表示**  
  どの曲が何回演奏されたかを集計できます。
- **マトリクス表示**  
  「どの公演で、どの曲が演奏されたか」を表形式で確認できます。
- **2days比較**  
  同じ会場の1日目/2日目を比べて、共通曲・日替わり曲を把握できます。
- **追記・整形（入力補助）**  
  貼り付けたセットリストを整形し、既存データに追記・更新できます。
- **新規公演の追加**  
  日付・会場情報を入力して、公演データを追加できます。

## 2. 使い方

### 2-1. アプリを開く

1. このリポジトリの `index.html` をブラウザで開きます。  
2. 画面上部のタブ（Dashboard / 公演別 / 曲別ランキング など）をクリックして切り替えます。

### 2-2. 基本操作

- **Dashboard**: 登録公演数、曲数、最終更新日、上位曲を確認
- **公演別**: 公演を選択して、曲順一覧と前公演との差分を確認
- **曲別ランキング**: 出現回数順で曲を確認
- **マトリクス**: 公演×曲の対応を表で確認
- **1日目/2日目比較**: 2daysの違いを確認
- **追記・整形**: 貼り付けテキストを整形し、保存

### 2-3. データ更新（追記・整形）の流れ

1. 「追記・整形」タブを開く
2. 対象公演を選ぶ
3. セットリストをテキストエリアに貼り付ける
4. 「整形する」を押す
5. 問題なければ「保存（追記・更新）」または「この公演を全上書き」を押す

## 6. 注意点

- このアプリで「保存」した内容は、**そのブラウザの localStorage にのみ保存**されます。  
  別の端末や別ブラウザには自動で同期されません。
- ブラウザの履歴削除・サイトデータ削除を行うと、保存内容が消える場合があります。
- `index.html` の中に初期データが入っているため、データ更新時は編集ミスに注意してください。
- まずはREADME更新のような**非破壊の変更**から始めるのは、とても良い練習方法です。
